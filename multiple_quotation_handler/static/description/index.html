Multiple Quotation Handler

Manage the Multiple Quotations In a Single Quotation.

Author : Nikhil krishnan

Features:

✍; Create Multiple Quotations for the same customer.
✍ Confirm the best quotation.
✍ Take print out of the multiple quotations.
✍ Email Template Contains The Multiple Quotations.

✎ Overview

In some business flows, We need to sent multiple quotations for the same customers, they choose the best for them. The clients will require negotiations whenever we sent a quotation, or need to change the products at the best price. For that, we need send the multiple products at the different rate and quality for the same use. In such cases, we can overcome the situation with this Multiple Quotations. Also, we can take the print Out of the multiple quotations and also we can send Email with multiple quotation email templates. For best Negotiations with clients, we can send multiple Quotations at a time and convert the best Quotation into Sale Order.
